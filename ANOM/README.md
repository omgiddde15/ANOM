# ANOM AI вЂ“ AI Powered Meaningful Connections Platform

ANOM AI is an AI-powered social networking platform that helps users build genuine and meaningful connections instead of endless scrolling. Using IBM Cloud technologies and Artificial Intelligence, the platform analyzes user profiles, recommends compatible people, generates conversation starters, and assists in planning safe meetups.

---

## Project Overview

Traditional social networking applications focus on engagement rather than meaningful relationships. ANOM AI solves this problem by combining Artificial Intelligence with interest-based matching and personality analysis to connect like-minded people.

The application provides an intelligent and secure environment where users can discover compatible people, communicate effectively, and schedule real-world meetings.

# Installation

## Clone Repository

```bash
git clone https://github.com/omgidde15/ANOM.git
```

## Install Frontend

```bash
cd client
npm install
npm run dev
```

## Install Backend

```bash
cd server
npm install
npm start
```

---

# Environment Variables

Create a `.env` file inside the server folder.

```
PORT=5000

JWT_SECRET=your_secret

CLOUDANT_URL=your_cloudant_url

CLOUDANT_APIKEY=your_cloudant_apikey

IBM_API_KEY=your_ibm_api_key

IBM_PROJECT_ID=your_project_id

IBM_URL=your_ibm_url
```
